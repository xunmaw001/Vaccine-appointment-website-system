# Vaccine-appointment-website-system
这是一款基于微信小程序的疫苗预约网站系统，依托微信生态，无需下载额外APP，打开微信即可轻松使用。系统清晰展示各类疫苗信息，包括种类、适用人群、接种禁忌等。用户能便捷地按时间和地点，选择合适接种点预约。还设有预约提醒，防止错过接种。管理端可实时监控库存、安排接种点工作、审核预约，为疫苗接种工作高效开展与公众健康提供有力保障 。
